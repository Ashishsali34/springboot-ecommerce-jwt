# 🛒 Spring Boot Ecommerce JWT Application

A production-ready ecommerce backend application built using Spring Boot and JWT Authentication.

## 🚀 Features
- JWT Authentication
- Role-Based Authorization
- User Management
- Product Management
- Category Management
- Cart Management
- Order Processing
- REST APIs
- Secure Authentication

## 🛠 Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- MongoDB / PostgreSQL
- Maven

## 📂 Project Structure
- Controller Layer
- Service Layer
- Repository Layer
- Security Layer
- Config Layer

## ▶ Run Project

Clone repository:

```bash
git clone https://github.com/Ashishsali34/springboot-ecommerce-jwt.git
```

Run:

```bash
mvn spring-boot:run
```
